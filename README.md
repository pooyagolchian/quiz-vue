# Vue js - Quiz application
* Build with Vue js
* You can see this demo on this [URL](https://pooyagolchian.ir/quiz-vue/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
